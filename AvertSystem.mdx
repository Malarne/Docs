Le principal outil **unique à AyRobot** pour la modération !

# Configuration:

- **avert config**: Affiche le menu de configuration de l'AvertSystem.
- **avert config general <channel>**: Choix du channel général du serveur. Utilisé uniquement pour les messages "Ban Hammer has spoken" des avert 3 et 4.
- **avert config invite <invite>**: Enregistrement d'une invitation à revenir sur le serveur envoyée lors de kick/unban si autorisé par l'utilisateur concerné.
- **avert config logs <channel>**: Choix du channel où seront envoyés les logs de modération.
- **avert config mod <channel>**: Choix du channel modérateur.
- **avert config muted <role>**: Choix du role attribué au membre lors de l'utilisation du **.mban**.
- **avert config report <channel>**: Choix du channel où seront envoyé les **.report** des membres.
- **avert config roleid <id>**: ID du role à tag lors d'un report.
- **avert config serveur <id>**: ID du serveur courant.
